# Basic Input & Output

This section contains Java programs that demonstrate basic input and output operations.

## Programs

### 1. Greeting.java
A simple program that reads the user's name and prints a personalized greeting on the screen.

**Features:**
- Uses Scanner class to read user input
- Demonstrates string concatenation
- Shows basic output using System.out.println()

### 2. ArithmeticOperations.java
A program that accepts two integers and displays their sum, difference, product, and quotient.

**Features:**
- Performs basic arithmetic operations
- Handles division by zero error
- Uses integer and floating-point arithmetic

### 3. WordCounter.java
A program that reads a full sentence and counts the number of words.

**Features:**
- Uses String.trim() and String.split() methods
- Handles edge cases like empty strings
- Counts words based on whitespace separation

### 4. VowelConsonant.java
A program that reads a single character and states whether it is a vowel or consonant.

**Features:**
- Character validation and classification
- Case-insensitive comparison
- Handles non-alphabetic characters

### 5. SalesComparison.java
A small shop wants a quick way to know which of three daily sales amounts is highest.

**Features:**
- Compares three integer values
- Uses conditional statements to find maximum
- Provides clear output of the result

## How to Run
To compile and run any of these programs, use the following commands in the terminal:

```bash
javac ProgramName.java
java ProgramName
```

Replace `ProgramName` with the actual name of the Java file (without the .java extension).
